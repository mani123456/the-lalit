== Description ==

Open Graph Protocol in Posts and Pages (OGP) allows you to add the Open Graph meta tags on a per page basis, or use default settings across the site.

= What is the Open Graph Protocol = 
The Open Graph protocol enables any web page to become a rich object in a social graph. For instance, this is used on Facebook to allow any web page to have the same functionality as any other object on Facebook.

= How does this plugin help? = 
This plugin add in an option page for setting your default meta to be used across your web site and also adds in meta boxes for posts, pages, media and public custom post types allowing you to customise what is shown in the open graph tags on each page.

The plugin also allows you to add in your own custom open graph tags if you so wish.

There is also filters available to customize the exact content shown for developers to use.

== Installation ==

1. Extract the zip file and drop the contents in the wp-content/plugins/OGP directory of your WordPress installation
1. Activate the Plugin from Plugins page
